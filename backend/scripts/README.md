# Add Stall Menus Script

This script adds menu items for different stalls to the database.

## How to Run

1. Make sure your backend server is not running
2. Navigate to the backend directory:
   ```bash
   cd backend
   ```

3. Run the script:
   ```bash
   node scripts/addStallMenus.js
   ```
